# BabyCare Web

React + TypeScript + Vite 기반의 웹 열람/공유 UI 시작점입니다.

## 실행

```bash
cd web
npm install
npm run dev
```

## 포함 화면

- 대시보드
- 사진첩
- 성장 타임라인
- 리포트
- 가족 공유 관리


## 폴더 구조

- `src/app`: 앱 엔트리와 셸
- `src/api`: 화면별 API/데이터 접근 계층
- `src/pages`: 화면 단위 컴포넌트
- `src/components`: 재사용 UI/레이아웃 컴포넌트
- `src/types`: 도메인/뷰 모델 타입
